# Personal Resume Website

🚀 **从零实现**的现代化个人简历网站模板，专为GitHub Pages部署设计。

## ✨ 项目特色

- 💯 **从零手写** - 完全自主开发，无任何第三方框架依赖
- 🆓 **完全免费** - 开源免费使用，可用于个人和商业项目
- 🎨 支持明暗主题切换
- 📱 完全响应式设计，适配各种设备
- 📝 数据驱动，所有内容通过`data.js`文件管理
- 🖨️ 打印优化样式
- 🌐 支持部署到GitHub Pages

⭐ **如果这个项目对你有帮助，请给个Star支持一下！**

## 🚀 快速开始

1. **Fork本项目**到你的GitHub账户
2. 编辑`data.js`文件中的个人信息
3. 添加个人图片到`images`文件夹
4. 根据需要修改`styles.css`中的主题颜色
5. 部署到GitHub Pages或任何静态网站托管服务

## 📁 文件说明

- `index.html` - 主页面结构
- `styles.css` - 样式和主题定义  
- `render.js` - 页面渲染逻辑
- `data.js` - 简历数据（主要编辑文件）
